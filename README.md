# 🏋️‍♂️ MuscleForge - Modern Fitness Landing Page

MuscleForge একটি আধুনিক এবং রেসপনসিভ জিম বা ফিটনেস ল্যান্ডিং পেজ যা Next.js এবং Tailwind CSS দিয়ে তৈরি। এই প্রজেক্টে একটি কমপ্লিট অথেন্টিকেশন সিস্টেম (Login & Registration) ইন্টিগ্রেট করা হয়েছে যা কাস্টম এপিআই (API) এর মাধ্যমে কাজ করে।

---

## ✨ Features
- **Modern UI/UX:** ল্যান্ডিং পেজে মাস্ক ইমেজ এবং আধুনিক টাইপোগ্রাফি ব্যবহার করা হয়েছে।
- **Full Authentication:** সাকসেসফুল রেজিস্ট্রেশন, ওটিপি (OTP) ভেরিফিকেশন এবং লগইন সিস্টেম।
- **Responsive Design:** মোবাইল, ট্যাবলেট এবং ডেস্কটপ সব ডিভাইসের জন্য অপ্টিমাইজড।
- **Interactive UI:** Framer Motion ব্যবহার করে স্মুথ অ্যানিমেশন যুক্ত করা হয়েছে।
- **Real-time Feedback:** API রেসপন্সের উপর ভিত্তি করে Hot-Toast নোটিফিকেশন।

---

## 🛠️ Tools & Packages Used

প্রজেক্টটি তৈরিতে নিচের টেকনোলজি এবং লাইব্রেরিগুলো ব্যবহার করা হয়েছে:

- **Framework:** [Next.js 16 (Beta/Latest)](https://nextjs.org/)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **API Handling:** [Axios](https://axios-http.com/) (v1.14.0)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) (v12.38.0)
- **Icons:** [React Icons](https://react-icons.github.io/react-icons/) (v5.6.0)
- **Notifications:** [React Hot Toast](https://react-hot-toast.com/) (v2.6.0)

## 🚀 Getting Started

প্রজেক্টটি আপনার লোকাল মেশিনে রান করতে নিচের ধাপগুলো অনুসরণ করুন:

### ১. প্রজেক্ট ক্লোন করুন
```bash
git clone [https://github.com/Arifuzzaman01/gym-center]
cd gym-center

ডেভেলপমেন্ট সার্ভার রান করুন
Bash
npm run dev
# অথবা
yarn dev
এখন ব্রাউজারে http://localhost:3000 ওপেন করে প্রজেক্টটি দেখুন।
